Syntax

Basic Syntax

if (condition) {
    // runs only if true
    System.out.println("Yes!");
}

The condition must be inside ( ) and the body inside { }.

üîÄ Types of if constructs

Runs a block only when condition is true. If false, nothing happens.

int marks = 75;
if (marks >= 40) {
    System.out.println("You passed!");
}

Provides an alternative ‚Äî one block for true, another for false.

int marks = 35;
if (marks >= 40) {
    System.out.println("Passed!");
} else {
    System.out.println("Failed!");
}

Checks multiple conditions one by one ‚Äî first true one wins.

int marks = 82;
if (marks >= 90) {
    System.out.println("Grade A");
} else if (marks >= 75) {
    System.out.println("Grade B");
} else if (marks >= 60) {
    System.out.println("Grade C");
} else {
    System.out.println("Grade D");
}

An if statement inside another if statement ‚Äî for complex conditions.

int age = 20;
boolean hasID = true;

if (age >= 18) {
    if (hasID == true) {
        System.out.println("Entry allowed");
    } else {
        System.out.println("No ID, denied");
    }
} else {
    System.out.println("Too young");
}
